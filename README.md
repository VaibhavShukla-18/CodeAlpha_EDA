# CodeAlpha EDA Project 🚢

## 📌 Project Title
Exploratory Data Analysis and Data Visualization on Titanic Dataset

## 🎯 Objective
To analyze the Titanic dataset and uncover patterns related to survival using data cleaning, visualization, and correlation analysis.

## 🛠 Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Tasks Performed
- Data loading and inspection
- Handling missing values
- Survival count analysis
- Gender-based survival comparison
- Age distribution visualization
- Correlation heatmap analysis

## 📈 Key Insights
- Female passengers had a significantly higher survival rate.
- Higher class passengers were more likely to survive.
- Age and fare showed moderate correlation with survival.
